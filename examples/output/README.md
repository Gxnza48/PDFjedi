# Example output

This folder is the **expected output** of running the PDFjedi pipeline against the synthetic input in [`../input/`](../input/).

The bundle is realistic in tone, content, and citation discipline. It demonstrates:

- How contradictions across PDFs are surfaced (see `99-open-questions.md`).
- How vague brief language ("modern and professional", "scalable") becomes concrete decisions plus open questions.
- How the budget, proposal, and contract stay aligned numerically and structurally.
- How citations work — every claim points back to a source PDF and page.
